# Genres.Attributes

The attributes for a genre resource.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### name

- **Type:** `string`
- **Required:** Yes

The localized name of the genre.

### parentId

- **Type:** `string`
- **Required:** No

The parent ID of the genre.

### parentName

- **Type:** `string`
- **Required:** No

The name of this genre’s parent genre.

### url

- **Type:** `string`
- **Required:** Yes

The clean URL for the genre.
