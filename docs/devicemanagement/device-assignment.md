# Device assignment

Manage devices for your students and employees.

## Topics

### Authentication

- [Authenticating for Automated Device Enrollment](/documentation/devicemanagement/authenticating-for-automated-device-enrollment) - Communicate securely with the web service using a server token.
- [Authenticating through web views](/documentation/devicemanagement/authenticating-through-web-views) - Use your own custom web interfaces to authenticate users.

### Device management

- [Activation Lock a Device](/documentation/devicemanagement/activation-lock-devices) - Enable activation lock on a remote device.
- [Get Device Details](/documentation/devicemanagement/device-details) - Get the details on a set of devices.
- [Get a List of Devices](/documentation/devicemanagement/fetch-devices) - Get a list of devices that are managed by the server.
- [Sync the List of Devices](/documentation/devicemanagement/sync-devices) - Get updates about the list of devices the server manages.
- [Disown Devices](/documentation/devicemanagement/disown-devices) - Notify Apple’s servers that your organization no longer owns the specified devices.
- [Get Replacement Details](/documentation/devicemanagement/get-replacement-details) - Get information about the device that a replacement device replaces.
- [Get Beta Enrollment Tokens](/documentation/devicemanagement/get-beta-enrollment-tokens) - Retrieves the beta enrollment tokens available for the organization.

### Profile management

- [Define a Profile](/documentation/devicemanagement/define-profile) - Define a profile that can be distributed to the devices in your organization.
- [Get a Profile](/documentation/devicemanagement/fetch-profile) - Get details about a profile.
- [Assign a Profile](/documentation/devicemanagement/assign-profile) - Assign a profile to a list of devices.
- [Remove a Profile](/documentation/devicemanagement/clear-device-profile) - Remove a profile from a list of devices.

### Account-driven enrollment service discovery

- [Assign Account-Driven Enrollment Service Discovery](/documentation/devicemanagement/assign-account-driven-enrollment-profile) - The Account-Driven Enrollment profile defines key attributes related to service discovery for account-driven enrollment by MDM.
- [Fetch Account-Driven Enrollment Service Discovery](/documentation/devicemanagement/fetch-account-driven-enrollment-profile) - Fetch the Account-Driven Enrollment profile that the MDM server sets, which includes information about service discovery for account-driven enrollment.
- [Remove Account-Driven Enrollment Profile](/documentation/devicemanagement/remove-account-driven-enrollment-profile) - Remove the Account-Driven Enrollment profile that the MDM server sets, which includes information about service discovery for account-driven enrollment.

### Objects and data types

- [Device](/documentation/devicemanagement/device) - A device’s properties and their values.
- [MachineInfo](/documentation/devicemanagement/machineinfo) - A device’s information in response to a MDM enrollment profile request.
- [Profile](/documentation/devicemanagement/profile) - A profile’s properties and their values.
- [Limit](/documentation/devicemanagement/limit) - A ranged limit.
- [Url](/documentation/devicemanagement/url) - A URL object.
