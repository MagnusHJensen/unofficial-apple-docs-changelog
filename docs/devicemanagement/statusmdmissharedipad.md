# StatusMDMIsSharedIPad

The status item that reports the device’s Shared iPad state.

**Platforms:** iOS 27.0 (Beta), iPadOS 27.0 (Beta), Mac Catalyst 27.0 (Beta)

## Properties

### mdm.is-shared-ipad

- **Type:** `boolean`
- **Required:** Yes

If `true`, the device is a Shared iPad.
