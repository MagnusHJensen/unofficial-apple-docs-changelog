# Books

A resource object that represents a book.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### attributes

- **Type:** `Books.Attributes`
- **Required:** No

The attributes for the books resource type.

### href

- **Type:** `string`
- **Required:** Yes

A relative location for the books resource.

### id

- **Type:** `string`
- **Required:** Yes

The identifier for the books resource.

### relationships

- **Type:** `Books.Relationships`
- **Required:** No

The relationships from books to other resources.

### type

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `books`

The type of the resource. The only allowed value is `books`.
