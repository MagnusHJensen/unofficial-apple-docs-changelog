# SettingsResponse

A response from the device after it processes the command to configure settings on a device.

**Platforms:** iOS 5.0, iPadOS 5.0, macOS 10.9, tvOS 9.0, visionOS 1.1, watchOS 10.0

## Topics

### Objects

- [SettingsResponse.ErrorChainItem](/documentation/devicemanagement/settingsresponse/errorchainitem) - A dictionary that describes an error chain item.
- [SettingsResponse.Settings](/documentation/devicemanagement/settingsresponse/settings-data.dictionary) - A dictionary that describes the results of configuring settings on a device.
