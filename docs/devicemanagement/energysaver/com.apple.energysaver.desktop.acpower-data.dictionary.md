# EnergySaver.Com.apple.EnergySaver.desktop.ACPower

The desktop AC power Energy Saver settings.

**Platforms:** macOS 10.7

## Properties

### Automatic Restart On Power Loss

- **Type:** `integer`
- **Required:** No
- **Allowed Values:** `0`, `1`

If `true`, enables “Start up automatically after a power failure.”

### Disk Sleep Timer

- **Type:** `integer`
- **Required:** No

The disk sleep time, in minutes. A value of 0 means never.

### Display Sleep Timer

- **Type:** `integer`
- **Required:** No

The display sleep time, in minutes. A value of 0 means never.

### Dynamic Power Step

- **Type:** `integer`
- **Required:** No
- **Allowed Values:** `0`, `1`

May not be available on all systems.

### Reduce Processor Speed

- **Type:** `integer`
- **Required:** No
- **Allowed Values:** `0`, `1`

May not be available on all systems.

### System Sleep Timer

- **Type:** `integer`
- **Required:** No

System sleep time, in minutes. A value of 0 means never.

### Wake on LAN

- **Type:** `integer`
- **Required:** No
- **Allowed Values:** `0`, `1`

If `true`, enables “Wake for network access.”

### Wake On Modem Ring

- **Type:** `integer`
- **Required:** No
- **Allowed Values:** `0`, `1`

If `true`, enables “Wake for modem ring.”
