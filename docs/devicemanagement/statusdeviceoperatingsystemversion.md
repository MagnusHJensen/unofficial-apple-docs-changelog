# StatusDeviceOperatingSystemVersion

A status report of the device’s operating system version.

**Platforms:** iOS 15.0, iPadOS 15.0, macOS 13.0, tvOS 16.0, visionOS 1.1, watchOS 10.0

## Properties

### device.operating-system.version

- **Type:** `string`
- **Required:** Yes

The operating system’s version in use on the device, such as `15.0`.
