# StatusDeviceOperatingSystemSupplementalExtraVersion

A status report of the device’s operating system’s Background Security Improvement identifier.

**Platforms:** iOS 16.1, iPadOS 16.1, macOS 13.0, tvOS 16.1, visionOS 1.1, watchOS 10.0

## Properties

### device.operating-system.supplemental.extra-version

- **Type:** `string`
- **Required:** Yes

The operating system’s Background Security Improvement version in use on the device, for example, `a`.
