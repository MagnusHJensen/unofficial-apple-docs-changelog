# StatusAccountListLDAP

The status item that lists the devices’s Lightweight Directory Access Protocol (LDAP) accounts.

**Platforms:** iOS 16.0, iPadOS 16.0, Mac Catalyst 16.0, macOS 13.0, visionOS 1.1

## Properties

### account.list.ldap

- **Type:** `[StatusAccountListLDAPAccountObject]`
- **Required:** Yes

A list of status values for the LDAP accounts.
