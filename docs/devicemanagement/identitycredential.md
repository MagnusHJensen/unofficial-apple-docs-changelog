# IdentityCredential

The data for a PKCS #12 password-protected identity.

**Platforms:** iOS 17.0, iPadOS 17.0, macOS 14.0, tvOS 17.0, visionOS 1.1, watchOS 10.0

## Properties

### Identity

- **Type:** `string`
- **Required:** Yes

The PKCS #12 identity data.

### Password

- **Type:** `string`
- **Required:** Yes

The password required to decrypt the PKCS #12 identity data.
