# WebContentFilterPluginFilter_BrowsersObject

Settings that control the browser filter. If not present, the system doesn’t use browser filtering.

**Platforms:** iOS 27.0 (Beta), iPadOS 27.0 (Beta), Mac Catalyst 27.0 (Beta), visionOS 27.0 (Beta)

## Properties

### Enabled

- **Type:** `boolean`
- **Required:** Yes

If `true`, the system enables filtering WebKit traffic.
