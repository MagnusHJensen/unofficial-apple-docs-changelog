# BaseRosterCourse

A base course’s properties and their values.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### name

- **Type:** `string`
- **Required:** No

The course name. The maximum length is 1024 UTF-8 characters.

### unique_identifier

- **Type:** `string`
- **Required:** No

The global unique identifier for the location. The maximum length is 256 UTF-8 characters.
