# App and book management (Legacy)

Manage apps and books for your students and employees.

## Topics

### Essentials

- [Getting app and book information (Legacy)](/documentation/devicemanagement/getting-app-and-book-information-legacy) - Use a web service to find details about apps and books to show to your users.
- [Managing apps and books through web services (Legacy)](/documentation/devicemanagement/managing-apps-and-books-through-web-services-legacy) - Associate volume purchases with users or devices using endpoints provided by the Volume Purchase Program (VPP).
- [Upgrading to Apple School Manager and Apple Business Manager](/documentation/devicemanagement/upgrading-to-apple-school-manager-and-apple-business-manager) - Manage devices and content across an organization’s user base with a single destination.

### Configuration management

- [Client Configuration](/documentation/devicemanagement/client-configuration) - Store client-specific information on the server.
- [Service Configuration](/documentation/devicemanagement/service-configuration) - Provides the full list of web service URLs and a list of possible error numbers.

### Asset and license management

- [Get Assets](/documentation/devicemanagement/get-assets-44p83) - Get the set of assets managed by your organization.
- [Get Assignments](/documentation/devicemanagement/get-assignments-158kc) - Get a list of assignments currently assigned to a user or device.
- [Get Licenses](/documentation/devicemanagement/get-licenses) - Get the set of licenses managed by your organization.
- [Manage Licenses](/documentation/devicemanagement/manage-licenses) - Associate and disassociate licenses with users and devices.

### User management

- [Get a User](/documentation/devicemanagement/get-a-user) - Get information about a particular user.
- [Get Users](/documentation/devicemanagement/get-users-5boi1) - Get information about a set of users.
- [Register a User](/documentation/devicemanagement/register-a-user) - Register a user with the volume-purchase program.
- [Edit a User](/documentation/devicemanagement/edit-a-user) - Modify details about a user.
- [Retire a User](/documentation/devicemanagement/retire-a-user) - Retire a user account.

### Objects and data types

- [VppAsset](/documentation/devicemanagement/vppasset) - A particular asset in the purchase program.
- [VppAssignment](/documentation/devicemanagement/vppassignment) - An assignment’s properties and their values.
- [VppLicense](/documentation/devicemanagement/vpplicense) - A license for a product in the purchase program.
- [VppAssociation](/documentation/devicemanagement/vppassociation) - An association between a license and a user or device.
- [VppUser](/documentation/devicemanagement/vppuser) - A user within the purchase program.
- [VppLocation](/documentation/devicemanagement/vpplocation) - A location used for managing purchases.
- [VppErrorCode](/documentation/devicemanagement/vpperrorcode) - An error code.
