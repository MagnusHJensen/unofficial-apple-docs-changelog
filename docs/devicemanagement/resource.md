# Resource

A resource such as an app or book.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### attributes

- **Type:** `Resource.Attributes`
- **Required:** No

The attribute metadata for the resource.

### href

- **Type:** `string`
- **Required:** No

The relative location for the resource, if it may be fetched directly.

### id

- **Type:** `string`
- **Required:** Yes

The identifier of the resource.

### meta

- **Type:** `Resource.Meta`
- **Required:** No

Contextual data about the resource.

### relationships

- **Type:** `Resource.Relationships`
- **Required:** No

The relationships for the resource.

### type

- **Type:** `string`
- **Required:** Yes

The type of the resource.

### views

- **Type:** `Resource.Views`
- **Required:** No

The views for the resource.
