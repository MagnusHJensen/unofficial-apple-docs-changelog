# WellKnown.AvailableServer

The configuration details for an authentication server.

**Platforms:** iOS 15.0, iPadOS 15.0, Device Assignment Services , VPP License Management 

## Properties

### BaseURL

- **Type:** `string`
- **Required:** Yes

The URL where the service resides. This is the base path for subsequent requests.

### Version

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `mdm-byod`, `mdm-adde`

The server’s version string.
