# WebContentFilter.WhitelistedBookmarksItem

The bookmark in the allow list of the web content filter.

**Platforms:** iOS 7.0, iPadOS 7.0, Mac Catalyst 7.0

## Properties

### Title

- **Type:** `string`
- **Required:** Yes

The title of the bookmark.

Deprecated: iOS 14.5+ | iPadOS 14.5+

### URL

- **Type:** `string`
- **Required:** Yes

The URL of the bookmark in the allow list.

Deprecated: iOS 14.5+ | iPadOS 14.5+
