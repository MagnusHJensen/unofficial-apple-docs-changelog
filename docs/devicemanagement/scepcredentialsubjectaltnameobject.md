# SCEPCredentialSubjectAltNameObject

The subject’s alternative name for the certificate.

**Platforms:** iOS 17.0, iPadOS 17.0, Mac Catalyst 17.0, macOS 14.0, tvOS 17.0, visionOS 1.1, watchOS 10.0, Device Assignment Services , VPP License Management 

## Properties

### dNSName

- **Type:** `string`
- **Required:** No

The DNS name.

### ntPrincipalName

- **Type:** `string`
- **Required:** No

The NT principal name. Use an other name OID set to `1.3.6.1.4.1.311.20.2.3`.

### rfc822Name

- **Type:** `string`
- **Required:** No

The RFC 822 email address.

### uniformResourceIdentifier

- **Type:** `string`
- **Required:** No

The uniform resource identifier.
