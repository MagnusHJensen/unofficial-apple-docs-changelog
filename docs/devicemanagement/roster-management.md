# Roster Management

Manage classes for your students and teachers.

## Overview

Administrators of Apple School Manager can use this API to access information about classes and people in their organization. If you’re a developer who needs access to information about somebody else’s organization, see the [Roster API](/documentation/RosterAPI).

> 

## Topics

### Account Management

- [Get Account Detail](/documentation/devicemanagement/account-detail) - Obtain the details for your account.

### Class Management

- [RosterClass](/documentation/devicemanagement/rosterclass) - A class’s properties and their values.
- [Get the List of Classes](/documentation/devicemanagement/fetch-class-roster) - Obtain a list of classes the server manages.
- [Sync the List of Classes](/documentation/devicemanagement/fetch-class-roster-sync) - Get updates about the list of classes the server manages.

### People Management

- [RosterPerson](/documentation/devicemanagement/rosterperson) - A person’s properties and their values.
- [Get the List of People](/documentation/devicemanagement/fetch-person-roster) - Obtain a list of people the server manages, across the organization.
- [Sync the List of People](/documentation/devicemanagement/fetch-person-roster-sync) - Get updates about the list of people the server manages.

### Course Management

- [BaseRosterCourse](/documentation/devicemanagement/baserostercourse) - A base course’s properties and their values.
- [RosterCourse](/documentation/devicemanagement/rostercourse) - A course’s properties and their values.
- [Get the List of Courses](/documentation/devicemanagement/fetch-course-roster) - Obtain a list of the courses the server manages.
- [Sync the List of Courses](/documentation/devicemanagement/fetch-course-roster-sync) - Get updates about the list of courses the server manages.

### Location Mangement

- [BaseRosterLocation](/documentation/devicemanagement/baserosterlocation) - A base location’s properties and their values.
- [RosterLocation](/documentation/devicemanagement/rosterlocation) - A location’s properties and their values.
- [Get the List of Locations](/documentation/devicemanagement/fetch-location-roster) - Obtain a list of the locations the server manages.
- [Sync the Locations](/documentation/devicemanagement/fetch-location-roster-sync) - Get updates about the list of locations the server manages.
