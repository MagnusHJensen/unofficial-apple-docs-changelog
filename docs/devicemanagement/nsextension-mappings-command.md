# NSExtension Mappings

Get a list of the installed extensions for a user on a device.

**Platforms:** macOS 10.13

## Discussion

This list is a superset of the list that [ActiveNSExtensionsCommand](/documentation/devicemanagement/activensextensionscommand) returns. It may contain extensions that the system never enables due to various restrictions.
