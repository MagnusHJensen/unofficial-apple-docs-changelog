# LoginItemsManagedItems.LoginItem

A dictionary with the details about a login item.

**Platforms:** macOS 10.13, Device Assignment Services , VPP License Management 

## Properties

### Hide

- **Type:** `boolean`
- **Required:** No
- **Default:** `false`

If `true`, the system hides this item in the Users & Groups login items list.

### Path

- **Type:** `string`
- **Required:** Yes

The URL or path string to the item’s location.
