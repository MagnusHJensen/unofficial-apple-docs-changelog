# Installing provisioning profiles on devices

Deploy provisioning profiles required to run enterprise apps on devices.

## Overview

Third-party enterprise apps require installation of a provisioning profile to enable running them. Use device management to efficiently deliver up-to-date versions of provisioning profiles. Using device management also prevents users from having to install provisioning profiles manually and handle replacement if they expire. For more information on installation, see [Install Provisioning Profile](/documentation/devicemanagement/install-provisioning-profile-command).
