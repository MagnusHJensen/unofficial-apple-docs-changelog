# ExtensibleSingleSignOn.PlatformSSO.TokenToUserMapping

The attribute mapping to use when creating users, or for authorization.

**Platforms:** macOS 14.0, Device Assignment Services , VPP License Management 

## Properties

### AccountName

- **Type:** `string`
- **Required:** No

The claim name to use for the user’s account name.

### FullName

- **Type:** `string`
- **Required:** No

The claim name to use for the user’s full name.
