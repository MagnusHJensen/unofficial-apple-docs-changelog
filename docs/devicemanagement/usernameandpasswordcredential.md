# UserNameAndPasswordCredential

Data that describes a credential that represents a user name and password.

**Platforms:** iOS 15.0, iPadOS 15.0, macOS 13.0, tvOS 16.0, visionOS 1.1, watchOS 10.0

## Properties

### Password

- **Type:** `string`
- **Required:** No

The password for this credential.

### UserName

- **Type:** `string`
- **Required:** Yes

The user name for this credential.
