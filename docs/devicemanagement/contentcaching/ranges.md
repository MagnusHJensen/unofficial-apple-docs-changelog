# ContentCaching.Ranges

A range of IP addresses to cache.

**Platforms:** macOS 10.13.4, Device Assignment Services , VPP License Management 

## Properties

### first

- **Type:** `string`
- **Required:** Yes

The first IP address in the range.

### last

- **Type:** `string`
- **Required:** Yes

The last IP address in the range.

### type

- **Type:** `string`
- **Required:** No
- **Default:** `IPv4`
- **Allowed Values:** `IPv4`, `IPv6`

The IP address type.
