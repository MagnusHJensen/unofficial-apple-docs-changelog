# EducationConfiguration.DepartmentsItem

A department in the organization.

**Platforms:** iOS 9.3, iPadOS 9.3, macOS 10.14

## Properties

### GroupBeaconIDs

- **Type:** `[integer]`
- **Required:** Yes

The group beacon identifiers that are members of this department.

### Name

- **Type:** `string`
- **Required:** Yes

The display name of the department.
