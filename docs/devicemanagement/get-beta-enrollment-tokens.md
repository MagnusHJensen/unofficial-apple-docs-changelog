# Get Beta Enrollment Tokens

Retrieves the beta enrollment tokens available for the organization.

**Platforms:** Device Assignment Services , VPP License Management 

## Topics

### Response

- [GetSeedBuildTokenResponse](/documentation/devicemanagement/getseedbuildtokenresponse) - Provides a list of beta enrollment tokens available for the given organization.
- [SeedBuildToken](/documentation/devicemanagement/seedbuildtoken) - Describes a beta enrollment token available for the given organization.
