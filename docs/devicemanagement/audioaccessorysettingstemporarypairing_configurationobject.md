# AudioAccessorySettingsTemporaryPairing_ConfigurationObject

A dictionary providing configuration for temporary pairing. Required if `Disabled` isn’t present or is `false`.

**Platforms:** iOS 26.0, iPadOS 26.0, Mac Catalyst 26.0, Device Assignment Services , VPP License Management 

## Properties

### UnpairingTime

- **Type:** `AudioAccessorySettingsTemporaryPairing_Configuration_UnpairingTimeObject`
- **Required:** Yes

A dictionary that describes when the device automatically unpairs temporarily paired audio accessories.
