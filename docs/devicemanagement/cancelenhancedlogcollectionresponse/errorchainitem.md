# CancelEnhancedLogCollectionResponse.ErrorChainItem

A dictionary that describes an error chain item.

**Platforms:** iOS 27.0 (Beta), iPadOS 27.0 (Beta), Mac Catalyst 27.0 (Beta), macOS 27.0 (Beta), tvOS 27.0 (Beta)

## Properties

### ErrorCode

- **Type:** `integer`
- **Required:** Yes

The error code.

### ErrorDomain

- **Type:** `string`
- **Required:** Yes

The error domain.

### LocalizedDescription

- **Type:** `string`
- **Required:** Yes

A description of the error in the device’s localized language.

### USEnglishDescription

- **Type:** `string`
- **Required:** No

A description of the error in U.S. English.
