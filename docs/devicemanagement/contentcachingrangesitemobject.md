# ContentCachingRangesItemObject

A range of IP addresses to cache.

**Platforms:** macOS 27.0 (Beta)

## Properties

### first

- **Type:** `string`
- **Required:** Yes

The first IP address in the range.

### last

- **Type:** `string`
- **Required:** Yes

The last IP address in the range.

### type

- **Type:** `string`
- **Required:** No
- **Default:** `IPv4`
- **Allowed Values:** `IPv4`, `IPv6`

The IP address type.
