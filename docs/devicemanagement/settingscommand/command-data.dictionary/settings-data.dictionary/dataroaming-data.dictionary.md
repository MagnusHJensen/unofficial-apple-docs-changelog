# SettingsCommand.Command.Settings.DataRoaming

A dictionary that contains data roaming settings.

**Platforms:** iOS 5.0, iPadOS 5.0

## Properties

### Enabled

- **Type:** `boolean`
- **Required:** Yes

If `true`, enable data roaming, which also enables voice roaming. If `false`, disable data roaming.

### Item

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `DataRoaming`

A string that identifies this setting.
