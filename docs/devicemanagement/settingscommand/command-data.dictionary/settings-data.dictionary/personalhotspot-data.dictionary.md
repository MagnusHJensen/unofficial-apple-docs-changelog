# SettingsCommand.Command.Settings.PersonalHotspot

A dictionary that contains Personal Hotspot settings.

**Platforms:** iOS 5.0, iPadOS 5.0, Mac Catalyst 5.0, Device Assignment Services , VPP License Management 

## Properties

### Enabled

- **Type:** `boolean`
- **Required:** Yes

If `true`, enable Personal Hotspot. If `false`, disable Personal Hotspot.

### Item

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `PersonalHotspot`

A string that identifies this setting.
