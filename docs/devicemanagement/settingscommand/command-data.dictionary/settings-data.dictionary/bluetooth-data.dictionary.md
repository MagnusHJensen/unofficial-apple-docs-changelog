# SettingsCommand.Command.Settings.Bluetooth

A dictionary that contains Bluetooth settings.

**Platforms:** iOS 11.3, iPadOS 11.3, Mac Catalyst 11.3, macOS 10.13.4, Device Assignment Services , VPP License Management 

## Properties

### Enabled

- **Type:** `boolean`
- **Required:** Yes

If `true`, enable the Bluetooth setting. If `false`, disable the Bluetooth setting.

### Item

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `Bluetooth`

A string that identifies this setting.
