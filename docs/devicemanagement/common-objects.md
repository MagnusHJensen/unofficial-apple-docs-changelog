# Common Objects

Understand the common JSON objects that framework responses contain.

## Topics

### Getting resource and relationship information

- [Resource](/documentation/devicemanagement/resource) - A resource such as an app or book.
- [Relationship](/documentation/devicemanagement/relationship) - A to-one or to-many relationship from one resource object to others.

### Getting common type information

- [Artwork](/documentation/devicemanagement/artwork) - An object that represents artwork.
- [DescriptionAttribute](/documentation/devicemanagement/descriptionattribute) - An object that represents a description attribute.
- [Genres](/documentation/devicemanagement/genres) - A resource object that represents a music genre.
- [Apps](/documentation/devicemanagement/apps) - A resource object that represents an app.
- [Books](/documentation/devicemanagement/books) - A resource object that represents a book.

### Handling errors

- [Error](/documentation/devicemanagement/error) - Information about an error that occurred while processing a request.
- [Interpreting HTTP status codes](/documentation/devicemanagement/interpreting-http-status-codes) - Interpret error codes the API might return when executing a request.
