# Active NSExtensions

Get a list of active extensions for a user on a device.

**Platforms:** macOS 10.13

## Discussion

This command returns information about the active extensions for a user. Extensions exist for each user, not for the device.

Extensions restricted from executing by Application Launch Restrictions or the [NSExtensionManagement](/documentation/devicemanagement/nsextensionmanagement) configuration profile wonâ€™t appear in the response.
