# IntelligenceSettingsApps_MailObject

If present, configures Mail Intelligence features.

**Platforms:** iOS 26.4 (Beta), iPadOS 26.4 (Beta), macOS 26.4 (Beta), visionOS 26.4 (Beta)

## Properties

### AllowSmartReplies

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Mail Smart Replies.

### AllowSummary

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Mail Summary.
