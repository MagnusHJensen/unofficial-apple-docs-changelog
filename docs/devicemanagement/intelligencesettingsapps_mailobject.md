# IntelligenceSettingsApps_MailObject

If present, configures Mail Intelligence features.

**Platforms:** iOS 26.4, iPadOS 26.4, macOS 26.4, visionOS 26.4

## Properties

### AllowSmartReplies

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Mail Smart Replies.

### AllowSummary

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Mail Summary.
