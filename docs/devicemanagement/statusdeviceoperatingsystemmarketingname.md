# StatusDeviceOperatingSystemMarketingName

The status item that reports the device’s operating system marketing name.

**Platforms:** iOS 15.0, iPadOS 15.0, Mac Catalyst 15.0, macOS 13.0, tvOS 16.0, visionOS 1.1, watchOS 10.0

## Properties

### device.operating-system.marketing-name

- **Type:** `string`
- **Required:** Yes

The operating system’s marketing name in use on the device, such as `Catalina`.
