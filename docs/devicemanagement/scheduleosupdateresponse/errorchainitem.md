# ScheduleOSUpdateResponse.ErrorChainItem

A dictionary that describes an error chain item.

**Platforms:** iOS 9.0, iPadOS 9.0, Mac Catalyst 9.0, macOS 10.11, tvOS 12.0, Device Assignment Services , VPP License Management 

## Properties

### ErrorCode

- **Type:** `integer`
- **Required:** Yes

The error code.

### ErrorDomain

- **Type:** `string`
- **Required:** Yes

The error domain.

### LocalizedDescription

- **Type:** `string`
- **Required:** Yes

A description of the error in the device’s localized language.

### USEnglishDescription

- **Type:** `string`
- **Required:** No

A description of the error in U.S. English.
