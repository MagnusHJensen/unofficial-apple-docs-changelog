# SkipKeys

The list of skip keys for setup panes.

**Platforms:** iOS 7.0, iPadOS 7.0, Mac Catalyst 7.0, macOS 10.9, tvOS 10.2, visionOS 26.0

## Properties

### Accessibility

- **Type:** `string`
- **Required:** No

The key to skip the Accessibility pane, when creating additional users.

> 

Available: macOS 11+
