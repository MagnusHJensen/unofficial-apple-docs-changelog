# ResponseSubscriptionAssignment

An assignment of a subscription to a user.

## Properties

### adamId

- **Type:** `int64`
- **Required:** No

The Adam ID for the subscription.

### clientUserId

- **Type:** `string`
- **Required:** No

The client user ID.

### renewing

- **Type:** `boolean`
- **Required:** No

A Boolean value that indicates whether the subscription is renewing.
