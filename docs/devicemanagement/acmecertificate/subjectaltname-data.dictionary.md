# ACMECertificate.SubjectAltName

The subject’s alternative name details.

**Platforms:** iOS 16.0, iPadOS 16.0, macOS 13.1, tvOS 16.0, visionOS 1.0, watchOS 9.0

## Properties

### dNSName

- **Type:** `string`
- **Required:** No

The DNS name.

### ntPrincipalName

- **Type:** `string`
- **Required:** No

The NT principal name. Use an other name OID set to `1.3.6.1.4.1.311.20.2.3`.

### rfc822Name

- **Type:** `string`
- **Required:** No

The RFC 822 (email address) string.

### uniformResourceIdentifier

- **Type:** `string`
- **Required:** No

The Uniform Resource Identifier.
