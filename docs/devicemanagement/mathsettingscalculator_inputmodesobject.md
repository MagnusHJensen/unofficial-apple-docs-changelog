# MathSettingsCalculator_InputModesObject

The declaration to configure the input modes in the calculator app.

**Platforms:** iOS 18.0, iPadOS 18.0, macOS 15.0

## Properties

### RPN

- **Type:** `boolean`
- **Required:** Yes

Configures whether RPN input is enabled.

### UnitConversion

- **Type:** `boolean`
- **Required:** Yes

Configures whether unit conversions are enabled.
