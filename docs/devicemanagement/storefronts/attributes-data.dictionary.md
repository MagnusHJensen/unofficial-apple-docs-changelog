# Storefronts.Attributes

The attributes for the storefronts resource.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### defaultLanguageTag

- **Type:** `string`
- **Required:** Yes

The default supported RFC 4646 language tag for the storefront.

### name

- **Type:** `string`
- **Required:** Yes

The localized name of the storefront.

### supportedLanguageTags

- **Type:** `[string]`
- **Required:** Yes

The supported RFC 4646 language tags for the storefront.
