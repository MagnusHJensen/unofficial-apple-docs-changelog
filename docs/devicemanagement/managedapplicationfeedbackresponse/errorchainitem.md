# ManagedApplicationFeedbackResponse.ErrorChainItem

A dictionary that describes an error chain item.

**Platforms:** iOS 7.0, iPadOS 7.0, Mac Catalyst 7.0, macOS 11.0, tvOS 10.2, visionOS 1.1, Device Assignment Services , VPP License Management 

## Properties

### ErrorCode

- **Type:** `integer`
- **Required:** Yes

The error code.

### ErrorDomain

- **Type:** `string`
- **Required:** Yes

The error domain.

### LocalizedDescription

- **Type:** `string`
- **Required:** Yes

A description of the error in the device’s localized language.

### USEnglishDescription

- **Type:** `string`
- **Required:** No

A description of the error in U.S. English.
