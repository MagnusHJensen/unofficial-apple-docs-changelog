# VPN.IPv4

The dictionary that contains IPV4 settings.

**Platforms:** iOS 4.0, iPadOS 4.0, macOS 10.7, visionOS 1.0

## Properties

### OverridePrimary

- **Type:** `integer`
- **Required:** No
- **Default:** `0`
- **Allowed Values:** `0`, `1`

If `1`, the system sends all network traffic over VPN. Only applies to Cisco IPsec and L2TP VPN types.
