# StatusDeviceOperatingSystemFamily

A status report of the device’s operating system family.

**Platforms:** iOS 15.0, iPadOS 15.0, Mac Catalyst 15.0, macOS 13.0, tvOS 16.0, visionOS 1.1, watchOS 10.0, Device Assignment Services , VPP License Management 

## Properties

### device.operating-system.family

- **Type:** `string`
- **Required:** Yes

The operating system family in use on the device, such as `macOS` or `iOS`.
