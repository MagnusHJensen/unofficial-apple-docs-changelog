# SCEP.PayloadContent.SubjectAltName

An optional dictionary that provides values required by the CA for issuing a certificate.

**Platforms:** iOS 4.0, iPadOS 4.0, Mac Catalyst 4.0, macOS 10.7, tvOS 9.0, visionOS 1.0, watchOS 3.0, Device Assignment Services , VPP License Management 

## Properties

### dNSName

- **Type:** `string`
- **Required:** No

The DNS name.

### ntPrincipalName

- **Type:** `string`
- **Required:** No

The NT principal name. Use an other name OID set to `1.3.6.1.4.1.311.20.2.3`.

### rfc822Name

- **Type:** `string`
- **Required:** No

The RFC 822 (email address) string.

### uniformResourceIdentifier

- **Type:** `string`
- **Required:** No

The Uniform Resource Identifier.
