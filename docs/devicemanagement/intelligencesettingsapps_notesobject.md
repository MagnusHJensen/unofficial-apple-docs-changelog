# IntelligenceSettingsApps_NotesObject

If present, configures Notes Intelligence features.

**Platforms:** iOS 26.4, iPadOS 26.4, macOS 26.4, visionOS 26.4

## Properties

### AllowTranscription

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Notes Transcription.

### AllowTranscriptionSummary

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Notes Transcription Summary.
