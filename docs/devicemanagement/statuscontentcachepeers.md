# StatusContentCachePeers

The status item that reports information about the Content Cache service peer caches.

**Platforms:** macOS 27.0 (Beta)

## Properties

### content-cache.peers

- **Type:** `[StatusContentCachePeersPeerItemObject]`
- **Required:** Yes

An array of dictionaries that describes peer Content Caches.
