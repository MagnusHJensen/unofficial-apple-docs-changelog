# GetAccountDrivenEnrollmentProfileResponse

The details for an account-driven enrollment profile.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### mdm_service_discovery_url

- **Type:** `string`
- **Required:** No

The MDM Service Discovery URL that the MDM server sets for redirection during account-driven enrollment.

### last_updated_timestamp

- **Type:** `string`
- **Required:** No

The timestamp of the most-recent update for the MDM Service Discovery URL.
