# Storefronts

A resource object that represents a region that the content is available in, and supported languages for that region.

**Platforms:** Device Assignment Services , VPP License Management 

## Properties

### attributes

- **Type:** `Storefronts.Attributes`
- **Required:** No

The attributes for the storefronts resource type.

### href

- **Type:** `string`
- **Required:** Yes

A relative location for the storefronts resource.

### id

- **Type:** `string`
- **Required:** Yes

The identifier for the storefronts resource.

### type

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `storefronts`

The type of the resource. The only allowed value is `storefronts`.
