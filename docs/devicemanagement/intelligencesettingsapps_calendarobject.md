# IntelligenceSettingsApps_CalendarObject

If present, configures Calendar and Reminders Intelligence features.

**Platforms:** iOS 26.4, iPadOS 26.4, Mac Catalyst 26.4, macOS 26.4, visionOS 26.4

## Properties

### AllowNaturalLanguageEditing

- **Type:** `boolean`
- **Required:** No
- **Default:** `true`

If `false`, disables Natural Language Editing in Calendar and Reminders.
