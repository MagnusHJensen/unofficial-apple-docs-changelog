# Dock.StaticItem

Items that are located on the Documents side of the Dock and cannot be removed from that location.

**Platforms:** macOS 10.7

## Properties

### tile-data

- **Type:** `Dock.StaticItem.Tile-data`
- **Required:** Yes

The information about the Dock item.

### tile-type

- **Type:** `string`
- **Required:** Yes
- **Allowed Values:** `file-tile`, `directory-tile`, `url-tile`

The type of tile.
